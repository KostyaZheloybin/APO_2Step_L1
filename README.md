# APO_2Step_L1

Результаты работы: 
Для расширения работы калькулятора был добавлен актор – OrepationActor, 
который хранит математическую операцию (произведение, сумма чисел, 
среднее значение, очистка списка, вывод списка) и отправляет сообщение 
актору калькулятор для выполнения математической операции.
